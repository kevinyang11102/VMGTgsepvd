# 人力资源管理系统 java580

## 项目概述


## 技术栈

- **后端：**
  - Spring Framework
  - Spring MVC
  - MyBatis

- **前端：**
  - Vue.js
  - Element UI

- **数据库：**
  - MySQL

## 功能模块

- **员工信息管理：**
  - 员工资料维护
  - 员工岗位调动

- **考勤管理：**
  - 考勤记录
  - 请假管理

- **薪酬管理：**
  - 工资单生成
  - 绩效考核

## 系统角色

- **系统管理员：**负责系统整体配置与维护
- **人事管理员：**负责员工信息管理与考勤记录
- **财务管理员：**负责薪酬核算与发放

## 运行环境

- **服务器：**
  - 操作系统：Linux或Windows
  - JDK版本：1.8及以上
  - 数据库：MySQL 5.7及以上

- **客户端：**
  - 现代浏览器

## 部署步骤

1. 准备运行环境
   - 安装JDK
   - 安装MySQL数据库
   - 部署应用服务器（如Tomcat）

2. 配置数据库
   - 初始化数据库
   - 导入初始数据

3. 部署后端服务
   - 部署后端代码
   - 启动服务

4. 部署前端
   - 部署Vue.js项目
   - 启动前端服务
